
## Installation

Install project

clone project 
```bash
  copy https://github.com/rawhx/pemogramanweblaravel
  cd pemogramanweblaravel
```
copy environment 
```bash
    copy .env.example .env
```
next install composer
```bash
  composer install
```
create key
```bash
  php artisan key:generate
```
migartion database
```bash
  php artisan migrate:fresh
```
run
```bash
  php artisan serve
```
